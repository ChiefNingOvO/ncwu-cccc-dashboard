# 天梯赛实时分数展示大屏

这是一个面向天梯赛/程序设计竞赛现场的实时分数展示系统，用于在 1920×1080 大屏上展示比赛队伍、队员成绩、队伍排名和题目得分变化。

项目包含前端大屏和本地后端服务：

- 前端使用 React、Vite、TypeScript 和 CSS Modules 构建。
- 后端使用 Express 提供 REST API。
- 数据层使用 sql.js 模拟 SQLite，用于保存比赛、队伍、队员和分数数据。
- 开发环境下 Vite 会把 `/api` 请求代理到后端服务 `http://localhost:3001`。

## 主要功能

- 比赛管理：首页展示比赛列表，支持创建、进入和管理比赛。
- 创建比赛：可以录入比赛名称、队伍和队员信息。
- 实时大屏：按固定 1920×1080 布局展示队伍排名、队员信息和详细题目得分。
- 队伍联动：点击左侧队伍列表后，右侧展示该队队员。
- 成绩详情：点击队员后显示 15 道题的分项成绩。
- 实时更新：内置模拟 WebSocket 分数推送，成绩变化时触发闪烁动画。
- 赛制规则：支持基础级、进阶级、登顶级三阶段题目与解锁门槛。

## 页面入口

| 路径 | 说明 |
| --- | --- |
| `/` | 比赛管理首页，展示所有比赛 |
| `/create` | 创建比赛页面 |
| `/live/:id` | 实时分数大屏 |
| `/live/:id/official` | 官方风格实时分数大屏 |

## 如何启动

### 1. 安装依赖

```bash
npm install
```

### 2. 启动后端服务

打开一个终端运行：

```bash
npm run dev:server
```

后端默认运行在：

```text
http://localhost:3001
```

首次启动时会自动创建演示比赛数据。

### 3. 启动前端开发服务

再打开一个终端运行：

```bash
npm run dev
```

前端默认运行在：

```text
http://localhost:5173
```

浏览器访问 `http://localhost:5173` 即可进入比赛管理首页。实时大屏建议使用 1920×1080 分辨率或浏览器全屏模式查看。

## 开发说明

开发时需要同时启动前端和后端：

- 前端端口：`5173`
- 后端端口：`3001`
- API 前缀：`/api`

前端请求 `/api/competitions` 等接口时，会通过 Vite 代理转发到后端。若只启动前端而没有启动后端，比赛列表、创建比赛和实时大屏数据加载会失败。

## 赛制规则

| 级别 | 题目数量 | 单题分值 | 解锁条件 |
| --- | --- | --- | --- |
| 基础级 | 8 | 5, 5, 10, 10, 15, 15, 20, 20 | 默认有效 |
| 进阶级 | 4 | 每题 25 分 | 队伍基础级总分不少于 800 |
| 登顶级 | 3 | 每题 30 分 | 队伍进阶级总分不少于 400 |

队伍总分由基础级、已解锁的进阶级和已解锁的登顶级成绩共同计算，并按总分实时排序。
